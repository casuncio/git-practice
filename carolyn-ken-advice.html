Hi, my name is Carolyn and I have some coding advice for you!

via GIPHY

  1. Code EVERYDAY.

    2. It doesn't matter if you spend 10 minutes and just write a few lines of code. This is a matter of building trust in yourself and discipline. You're learning a new way of thinking and if you step away from too long it will be hard to start back up - it's like running the mile in school. Even if you're running slow, keep going.

  2. Find a Good Teacher and a Supportive Community.

    3. Contrary to popular belief, coding is a collaborative activity (at least when you're learning). At some point you will get stuck and need questions answered. While it's great to Google things and that will get you far, you might get easily discouraged if you get stuck and can't find the answers so best have someone around to help! Skillcrush offers both these things!

  3. Be Kind to Yourself

    4. Coding can be hard, so don't be ashamed if you have to re-read things or back and re-watch videos or find and read and watch other sources. This isn't a race and the more you take time, concepts will solidify in your head.

  4. Practice, Practice Practice

    5. You need practice to get concepts to stick in your head. If there is additional practice, or optional projects, do them!

  5. Don't Give Up.

    6. In problem solving, you never know when the problem is just a missing ; or >! Keep going! It will be so satisfying when you find the answer.

You got this! Keep going and don't stop!
